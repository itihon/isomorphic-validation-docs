Creates a renderer function that renders the first validation error message of a validation result.

## Syntax

```js 
renderFirstError()
renderFirstError(errorPropName)
renderFirstError(errorPropName, toString)
```

### Parameters

- `errorPropName` <sup>optional</sup>
<br/> A property name of an error message supplied with a predicate function as `anyData` when adding as a [constraint](/api/validation/instance-methods/constraint/#parameters).
<br/> **Default**: `'msg'`.

- `toString` <sup>optional</sup>
<br/> A function that accepts a validation result entry, index, and an `Array` of the entries and returns a string.
<br/> **Default**: implementation: [render-entry.js](https://github.com/itihon/isomorphic-validation/blob/master/src/helpers/render-entry.js).

### Return value

A function that accepts a [`ValidationResult`](/api/validation-result/) object and returns a string. 

## Description

Creates a function that is intended to be used as the `value` property in the `stateValues` object which is passed as a parameter into the [`applyBox()`](/api/ui/effect-functions/apply-box/#parameters) function.

<br/>**Implementation:** [render-first-error.js](https://github.com/itihon/isomorphic-validation/blob/master/src/ui/render-first-error.js).
