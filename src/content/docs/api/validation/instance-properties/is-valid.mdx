The `isValid` accessor property returns the current validity state of the `Validation` object.

## Value

A `Boolean` value.

## Description

This property returns `true` when all predicate groups of the `Validation` are in the "valid" state. After creation of an instance of `Validation`, this property is set to `false`, except for validations created with the parameter [`optional=true`](/api/validation/constructor#parameters) for which it is set to `true` by default.
