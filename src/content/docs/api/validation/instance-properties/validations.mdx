The `Validation().validations` property returns an `Array` that contains grouped `Validation` objects.

## Value

An `Array` of `Validation` objects.

## Description

On a [single](/concept/single-validation) `Validation` object, this property returns an empty `Array`.  On a [grouping](/concept/grouping-validations) `Validation`, this property returns an `Array` of the grouped `Validation` objects which were passed in the [`Validation.group()`](/api/validation/static-methods/group) or [`Validation.glue()`](/api/validation/static-methods/glue) method.
