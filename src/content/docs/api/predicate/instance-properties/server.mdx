The `Predicate().server` property "marks" all the methods invokations following after it as being invoked on the server side only and ignored on the client side.

## Value

The `Predicate` object.

## Description

The purpose of this property is to provide a way of adding [state callbacks](/concept/state-callbacks/) to a `Predicate` object separately for the client and server side.
