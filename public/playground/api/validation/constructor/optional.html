"First name" and "Last name" fields should be filled in with only letter characters.

The "Age" field is optional and should be filled in with a natural number between 21 and 45.

The "Secret question" should be less than 250 characters long.

Note: There is a quirk in Google Chrome, Firefox and maybe in other browsers. If you type in an input field with type="number" something like "1234+" or "1234-" it transforms into an empty string, which is valid as a value for an "optional" field.
