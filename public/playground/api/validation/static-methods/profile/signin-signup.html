Login:
  • On both forms: Allowed characters are: letters, numbers, ".", "_", "-".
  • On both forms: Minimal length is 5 characters.
  • On both forms: Maximum length is 32 characters.
  • On the sign-up form: Login must not be already registered. ("admin" is registered)
Password:
  • On both forms: Should contain at least 1 capital letter, 1 number, and 1 special character ("!", "@", "#", etc).
  • On both forms: Minimal length is 8 characters.
  • On the sign-up form: Password and password confirmation must be the same.
