1. Default class names

Used the default class names "valid" and "invalid" wihout delay.

2. Specified class names

Used specified class names, delay is set for the invalid state.

3. Overriden target

The same effect as above is applied to the parent node.

4. Indicate only valid state

Highlights only when the state is valid, without delay.

5. Delayed effect

Different delays are applied for valid and invalid states.

6. The same effect ID

Different effects with the same effect ID cancel each other. Highlighting is being cleared on each keystroke.

7. Clear only invalid state

The same effect as above but highlighting is being cleared on each keystroke only for the invalid state.
